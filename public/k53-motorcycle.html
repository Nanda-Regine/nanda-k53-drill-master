Frequently asked questions

Does a motorcycle learner need to be supervised?

No. Unlike a car (Code 2/3) learner, a Code 1 motorcycle learner rides unsupervised — and may not carry a passenger, so supervision on the bike isn’t possible.

How old must I be for a motorcycle learner’s licence?

From 16 for a Code A1 (up to 125 cc) and from 18 for an unrestricted Code A motorcycle.

How many questions is the Code 1 test?

40 questions, and you need 75% in each section to pass.
